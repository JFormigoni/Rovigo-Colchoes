## Como Adicionar no Painel Admin

1. Acesse: `seu-site.web.app/admin/`
2. Faça login
3. Clique em "Adicionar Novo Produto"
4. Copie e cole os dados de cada produto
5. Marque os checkboxes conforme indicado
6. Clique em "Salvar Produto"

---

## Dicas

- Os produtos com "Promoção: Sim" mostrarão o preço riscado e o preço promocional em destaque
- Os produtos com "Destaque: Sim" aparecerão na página inicial
- O produto "Kids Comfort" está marcado como fora de estoque para demonstrar essa funcionalidade
- As URLs das imagens são do Unsplash (serviço gratuito de imagens)

## Testando o Fluxo Completo

Após adicionar os produtos:

1. Acesse a página inicial - deve ver os produtos em destaque
2. Acesse a página de produtos - deve ver todos os produtos
3. Clique em um produto
4. Selecione cor e tamanho
5. Clique em "Falar com Vendedor no WhatsApp"
6. Verifique se a mensagem foi gerada corretamente
